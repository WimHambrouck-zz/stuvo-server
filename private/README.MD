Drankregistratie
-------

URL

    api/drankregistratie.php


----------


Parameters:
**Email**
De email die je probeert te registreren

    email=maxim@example.com


----------


Response

**Indien OK**
```json
{status:"0"}
```
**Indien geen studenten email**
```json
{status:"1"}
```
**Indien reeds geregistreerd**
```json
{status:"2"}
```
